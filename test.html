Skills

I am constantly learning new skills and broaden my knowledge, below are among the technologies that I feel comfortable with.

PHP

PHP is my server-side language of choice. Having volunteered with charities' WordPress sites, it was important for me to understand PHP and having learnt OOP PHP 7 I understand the power that is available with PHP. I enjoy writing code in Vanilla PHP or using a framework such as Laravel.

WordPress

I have volunteered with two charities assisting with their WordPress development. Born to be Beautiful are based in Cambridge and assist women in India, Uganda and Sierra Leone to start a career as a beauty therapist. WordPress being the world leading CMS system, I knew it was important that I got some exposure to it in a professional environment.

Laravel

Laravel is my favourite PHP MVC framework. I have built a variety of projects with it including my travel blog site and I love the community. Using Laravel ensures that I write OOP code and assists making clean readable code.

Git

I track my code with the version control system, Git. It ensures that I can track changes that I have made to my code. It is also much easier for me to deploy any changes to my server.

Nginx

I have a VPS Nginx server that I use to host my projects. It runs on Ubuntu and I find that it is a quick and easy solution to add projects and modify existing ones.

MySQL

I use the MySQL database as my data source. Having previously worked with SQL Server whilst working in FP&A, I gained a good understanding of database modelling techniques, from my web development studies I gained an understanding of PHP's PDO statements and Laravel's Eloquent requests.
